## 简介
轻量级微服务架构 仿deepseek大模型对话 电脑端/移动端+管理端。<p/>

核心功能：流式对话/历史对话/内容下载/多平台模型切换(ollama、魔撘社区、阿里云百炼平台)/LLM应用中心

技术栈<br>
* Vue3、CSS、Element Plus、vant
* Spring Boot3
* Redis/MySQL/OSS
