## 简介
采用微服务架构，依托主流LLM应用平台，面向行业用户构建智能体SAAS应用服务中心。<p/>

核心功能<br>
* 支持双端：电脑端/移动端
* 大模型对话：流式对话/历史对话/内容下载
* 智能体：面向行业的智能体
* 支持主流大模型平台:ollama/魔撘社区/百炼平台等
* 支持LLM应用中心:百炼/Dify/N8N/FastGPT/钉钉等
* 基础能力：用户管理/租户管理/计费管理/智能体市场

技术栈<br>
* Vue3、CSS、Element Plus、vant
* Spring Boot3
* Redis/MySQL/OSS
